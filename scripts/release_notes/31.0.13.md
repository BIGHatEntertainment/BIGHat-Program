## Cloud Library / file-cloud sync removed

Premium content packs are now sold as `.bighat` files via Squarespace.
The file-cloud distribution feature has been retired:

* `/api/native/sync/*` endpoints removed.
* `cloud_sync_enabled` subscription flag removed (existing configs
  scrub it on next load).
* "Cloud Library" tier UI removed from the Setup Wizard.
* "Cloud Library / Local-only" trivia-source picker collapsed to local.
* SharePoint asset-factory now hard-coded to local-only.

What this does **not** change:

* License activation against `api.bighat.live` still works.
* Squarespace webhook → license key emails (Resend) still works.
* HWID + seat tracking still works.
* OS-aware `bighat.live/download` page still works.
* Your own host SharePoint pipeline (Round Maker upload, Slide Fetcher,
  Story Builds) is unchanged.

## .bighat file format unchanged from v31.0.12

Continue using Export / Import / Inspect endpoints for all four
content types: round, presentation, bingo, scoreboard. HMAC signing
remains available for premium pack distribution.
