## New: .bighat file format — share, back up, and sell content

Hosts can now export Round Maker rounds, full trivia presentations,
bingo cards, and scoreboard themes as portable `.bighat` files. Each
file is a single archive containing the content + any media assets
(cover images, winner videos, audio clips). Use them to:

* Email rounds to fellow hosts.
* Back up your library to OneDrive or a USB stick.
* Move from one PC to another without retyping anything.
* Buy and import premium content packs sold on bighat.live (coming
  soon).

### How to use it

* In Round Maker, Bingo Lobby, Trivia Dashboard, or Scoreboard Studio,
  click **Export** next to any item to download a `.bighat` file.
* Click **Import** at the top of any of those pages, pick a `.bighat`,
  and confirm the preview. The content lands in your library
  immediately — no internet required.
* Double-click any `.bighat` file in Windows Explorer to import it
  directly (file association registered by the installer).

### Premium content packs

Files can be cryptographically signed by the BIG Hat publisher.
Officially-released packs from BIG Hat Entertainment will display a
green "Signed by a verified BIG Hat publisher" badge in the import
dialog. Personal exports from your own install show an "Unsigned"
notice — both work the same way; the badge is informational.
