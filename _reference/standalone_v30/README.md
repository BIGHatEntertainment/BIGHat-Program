# BIG Hat Entertainment - Professional Hub

## 🚀 Installation Instructions
1. **UNZIP** this entire folder to your Desktop or Documents folder.
   *(The installer will not work correctly if run directly from inside the Zip)*
2. Double-click **SETUP_BIGHat_Entertainment.vbs**.

## 🛡️ What this does:
- Verifies you have **25GB of free space** for proprietary game data.
- Maps your system to **C:\BIG Hat**.
- Creates your **Desktop Shortcut** with the BIG Hat logo.
- Launches the **Proprietary Setup Wizard** to create your Master Admin account.
