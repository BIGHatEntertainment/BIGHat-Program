# BIG Hat Entertainment — DESKTOP runtime dependencies.
#
# Subset of requirements.txt baked into the embedded Python that ships
# with the Windows / macOS installer. Everything that's only used on the
# dev box (linters, type checkers) or only by the cloud license server
# (boto3, stripe, resend on the server side) is stripped here so we
# don't bloat the customer download.
#
# Pinned versions match requirements.txt exactly — DO NOT diverge, or
# the desktop wheels will resolve differently from what we test against
# on the dev box. When you bump a version in requirements.txt, mirror it
# here too.
#
# Build pipeline: scripts/build_installer.py + scripts/build_dmg.py
# call `pip download --platform … --only-binary=:all:` against this file
# to fetch native wheels for the customer's platform.

# crashes with System.NullReferenceException on some Win 11 installs. The
# launcher now spawns msedge.exe / chrome.exe with --app=URL directly,
# which gives us a chromeless Chromium window without any .NET binding.
# Saves ~25 MB off the installer too.
